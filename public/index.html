Placeholder image generator

This API provides you with the ability to generate a placeholder image of your style in the dimensions that's needed.

API routes:

/image/width/height

Example: 

/image/256/256

Returns an image with the size specified in the width and height section, and with default styling.

/image/width/height/backgroundColor

Example:

/image/256/256/orange
/image/256/256/hex:4545bb

Returns an image with the size specified in the width and height section, and with a custom background color. The color can either be a string (like "blue" and "red"), or a hex color prefixed with "hex:" and without the hashtag, such as "hex:fffbbb"

/image/width/height/backgroundColor/textColor

Example: 

/image/256/256/orange/blue
/image/256/256/red/hex:454545

Returns an image with the size specified in the width and height section, and custom colors for the background and the text. The colors can either be a string (like "blue" and "red"), or a hex color prefixed with "hex:" and without the hashtag, such as "hex:fffbbb"

Rules:

  • Width and height can be 2048px each maximum.
  • Hex colors has to be prefixed by "hex:" and have the hashtag removed.

Example:

/image/300/200


/image/600/600/orange/hex:fff
